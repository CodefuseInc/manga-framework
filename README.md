# Manga-framework
## Based on react

A eLearning Framework written in ReactJS

## Setup
1. `npm install`.


## Dev
1. `npm start` - Compile, start server, watch for file changes

## Test
1. `npm run test` - Run test suite

## Deploy
. `./node_modules/.bin/gulp compile deploy` - Compiles all assets into /dist;
